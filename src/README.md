# Source Code

This directory contains the Google Apps Script source files for the Lunch Order System.

## Files

- **Code.gs** - Google Apps Script backend (~1,800 lines)
- **WebApp.html** - Frontend HTML/CSS/JavaScript (~2,200 lines)

## Deployment

1. Create a new Google Apps Script project
2. Copy `Code.gs` content to `Code.gs` in the editor
3. Create `WebApp.html` file and paste content
4. Update `CONFIG.SPREADSHEET_ID` with your spreadsheet ID
5. Deploy as Web App

## Note

The full source code should be copied from your deployed Google Apps Script project or from the original source files provided with this repository.

If the files appear truncated in this repository, refer to the original deployment at your Google Apps Script project.
